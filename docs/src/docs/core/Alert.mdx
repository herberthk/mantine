## Usage

Use Alert to attract user attention with important message.
Alert is designed to be static, if you need more dynamic variant use <GatsbyLink to="/core/notification/">Notification</GatsbyLink> component.

## Color

You can choose any color defined in <GatsbyLink to="/pages/theming/">theme</GatsbyLink>. By default Alert will have `theme.primaryColor` color.

```tsx
<Alert /> // -> theme.primaryColor (defaults to blue)
<Alert color="red" /> // -> red color
```
