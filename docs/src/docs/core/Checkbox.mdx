## Usage

Use Checkbox to capture boolean value input from user.
For better accessibility set `label` prop, it will add associated label element.

<CheckboxDemos.configurator />

## Sizes

Checkbox has 5 predefined sizes: xs, sm, md, lg, xl.
Size defines label font-size (from <GatsbyLink to="/pages/theming/">theme.fontSizes</GatsbyLink>), input width and height.

```tsx
<Checkbox size="xl" /> // -> predefined xl size
```

## Indeterminate state

Checkbox supports indeterminate state. When `indeterminate` prop is set to true,
`checked` prop is ignored:

<CheckboxDemos.indeterminate />

## Controlled

```tsx
import React, { useState } from 'react';
import { Checkbox } from '@mantine/core';

function Demo() {
  const [checked, setChecked] = useState(false);
  return (
    <Checkbox checked={checked} onChange={(event) => setChecked(event.currentTarget.checked)} />
  );
}
```

## Get element ref

You can get input ref by setting `elementRef` prop:

```tsx
import React, { useRef } from 'react';
import { Checkbox } from '@mantine/core';

function Demo() {
  const ref = useRef();
  return <Checkbox elementRef={ref} />;
}
```

## Accessibility

Provide `aria-label` in case you use checkbox without label for screen reader support:

```tsx
<Checkbox label="My checkbox" />; // -> ok, input and label is connected
<Checkbox />; // not ok, input is not labeled
<Checkbox aria-label="My checkbox" />; // -> ok, label is not visible but will be announced by screen reader
```
