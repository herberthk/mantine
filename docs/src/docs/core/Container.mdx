## Usage

Container is the most basic layout element, it centers content horizontally and add horizontal padding from theme.
Component accepts these props:

- `size` – controls default max width
- `padding` – controls horizontal padding of container, pass `xs | sm | md | lg | xl` for value defined in `theme.spacing` or number to set horizontal padding in px
- `fluid` – overwrites `size` and sets max width to 100%
